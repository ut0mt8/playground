Plugin Playground

Plugin Playground is an open-source general-purpose runtime tweak system for macOS Apple Silicon.

It provides a kernel-level interception engine built on Mach exception ports and ARM64 hardware breakpoints for building plugins, introspection tools, and behavior-modification tweaks that work on any process.

Includes

bin/fangsCore runtime modifier
bin/grantInitializer for core runtime
Plugin Playground.appGUI for managing tweaks (in /Applications)
tweaks/User-owned directory for tweak bundles
